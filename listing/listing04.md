Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
дедлок. Потому что горутина не закрыла канал, а рэндж будет ждать значение канала, пока он не будет закрыт

```
